DESCRIPTION

This library supports efficient 2D Shape Algorithms:
  Spatial index, i.e., locator grid for quickly locating nearby shapes.
  Determine if two shapes overlap.
  Determining Shape-Shape intersection point.
  Find shortest line segment connecting two shapes.

BUILD

Requires cf01 library.

1. Start Cygwin
2. >cd np02/slickedit_gcc
3. >make --makefile=np02.mak CFG=Debug
4. >make --makefile=np02_test.mak CFG=Debug
(configurations = Debug, Release, or Profile)


TEST

1. Start Cygwin
2. >cd np02/slickedit_gcc/Debug
3. >./np02_test.exe --test=1 --test-rand-seed=17 --test-iterations=10
4. >./np02_test.exe --test=2 --test-rand-seed=43 --test-iterations=10
5. Open bitmap files found in same directory.
